# Invoicing Integration Documentation

Welcome to the documentation for the Invoicing Integration package. This documentation provides an overview of the features, usage, and API details for integrating with invoicing systems, with a focus on Cegid Vendus.

## Overview

This package helps you integrate with invoicing systems, allowing you to:

-   Create invoices and receipts
-   Assign clients, items, and payments
-   Configure provider-specific settings (e.g., Cegid Vendus)
