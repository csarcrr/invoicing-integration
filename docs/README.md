# Invoicing Integration - Documentation

<b>Invoicing Integration is a Laravel package that aims to help developers
integrate Portuguese invoicing systems into their app without have to force the developer
to understand every quirk of the invoicing system API.</b>

We are currently in a very Alpha stage, and we are trying to figure out the best ways to "translate"
the APIs in a more agnostic and unbiased approach.

The first provider we are starting to support is <a href="https://www.Cegid Vendus.pt/">Cegid Vendus</a>.
